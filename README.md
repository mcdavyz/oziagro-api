# OziAgro

## Overview

OziAgro is a Python-based decision support platform for precision agriculture.

The platform analyzes historical climate data to generate agroclimatic indicators and provide decision support for farmers, extension officers, and researchers.

## Current Features

- Read rainfall datasets
- Calculate growing season length
- Classify rainfall risk
- Generate processed datasets

## Technology Stack

- Python
- Pandas
