# Configuration

## Overview

Application settings are managed centrally through the configuration module.

Typical configuration includes:

- API title
- API version
- API description
- Upload directory
- Default climate dataset

---

## Environment

Current environment:

- Python
- FastAPI
- Pandas
- NumPy

---

## Upload Directory

Uploaded Excel files are temporarily stored in the configured upload directory before processing.

---

## Default Dataset

The sample analysis endpoint uses the configured default rainfall dataset.
